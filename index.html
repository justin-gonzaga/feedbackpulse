Assignment Specification

Temperature Converter

Create a program that converts temperatures between Celsius and Fahrenheit. The program should:

  1. Allow users to convert from Celsius to Fahrenheit
  2. Allow users to convert from Fahrenheit to Celsius
  3. Track conversion history
  4. Display a menu for users to select options
  5. Handle invalid inputs appropriately

Formulas to use:

  • Celsius to Fahrenheit: F = (C × 9/5) + 32
  • Fahrenheit to Celsius: C = (F - 32) × 5/9

Marking Rubric

Total: 25 points

Category Points Criteria
Functionality 6 Conversion Accuracy (C to F: 3, F to C: 3)
5 User Interface (Menu: 2, I/O clarity: 3)
4 History Feature (Storage: 2, Display: 2)
Code Quality 3 Structure (Functions: 2, Organization: 1)
3 Error Handling
4 Style (Variable names: 2, Comments: 2)

Assignment Specification

Restaurant Data Analysis

Write SQL queries to extract and analyze information from a restaurant database containing tables for menu items, orders, and customers.

Required Queries:

  1. List all menu items with a price over $7.50, ordered by price (descending)
  2. Find the total number of orders placed by each customer
  3. Calculate the average order amount
  4. Find the most popular menu item (most frequently ordered)
  5. List all orders placed on March 15, 2023 with their total amounts

Database Structure (given):

  • menu_items: id, name, price, category
  • customers: id, name, email
  • orders: id, customer_id, order_date, total_amount
  • order_items: order_id, menu_item_id, quantity

Marking Rubric

Total: 25 points

Category Points Criteria
Query Correctness 10 All queries return correct results
5 Proper use of joins and relationships
5 Correct filtering conditions
Efficiency 3 Efficient query structure
2 Proper use of indexes and query optimization
